# Proficiencies prompt

Añade una tabla debajo de las skills que tenga las competencias en Armas, Armaduras, Tools, Idiomas y Extras

Cuando clique a las armas debera aparecer 3 campos, Simples, Marciales y custom. Si el usuario clica en Simples o Marciales se le añade esa competencia (es decir, se va el pop up de lo btn y aparece en texto en la tabla, la que el usuario haya elegido) y luego si clican en custom, aparecera un campo de texto para que puedan poner el texto que ellos quieran.

En Armor un poco parecido, pero los campos seran Light, Medium, heavy y custom.
Tools, Languages y extras de momento que solo aparezca el btn de custom


# Module Prompt
Vamos a actualizar los modulos, cuando cliquen en add module, el usuario debera elegir si es una Acction, Bonus Acction, Reaction, Free Acction o Other. Por lo que en la top bar del Tabbed Panel ahora aparecera Featrures para todas las features, pero ahora podran filtrar segun el tipo. Los filtros seran del tipo selectivo, podran seleccionar accion y bonus accion, por lo que solo apareceran las que tengan esas caracteristicas.
